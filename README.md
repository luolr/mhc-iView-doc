## 基于iView的Vue.js组件库文档

[在线DEMO](https://luolr.github.io/mhc-iView-doc)

### 编译运行

``` bash
# 克隆项目
git clone https://github.com/luolr/mhc-iView-doc.git

# 安装依赖
npm install

# 本地运行
npm run dev
```
